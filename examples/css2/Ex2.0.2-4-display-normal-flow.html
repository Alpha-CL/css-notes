默认情况 width 自动吸收剩余横向距离
子项 设置 width: auto; && margin: auto; 时，width 权重较大，吸收剩余横向宽度
子项设置 width, border, padding 后，默认 margin-right 自动吸收剩余横向距离
子项设置 padding, margin, width 百分比时都是相对于父级的宽度
margin，padding，border 仅作用域水平空间占位
width: 300px; height: 400px
子项设置 padding, margin, width 百分比时都是相对于父级的宽度值进行计算 padding: 150px; margin: 150px
无论父子还是兄弟元素，margin 垂直方向相邻，即会合并( 仅较大的 margin值 有效 )
