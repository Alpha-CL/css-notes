#### nested properties

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 属性嵌套
 * 
 * 
 * 部分CSS属性遵循相同的命名空间( namespace )
 * eg: font-family, font-size, font-weight ...          font-xxx
 * 
 * 为了便于管理以相同命名空间开头的属性，可以将该部分属性嵌套在 相同的命名空间中
 * 
 */

.demo {

    font: {
    
        family: fantasy;
        size: 30em;
        weight: bold;
    }
}


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
