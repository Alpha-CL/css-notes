#### Animation

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * @keyframes frameName{ 0% | from {} ... xx% {} ... 100% | to {} }
 * 
 * 指定动画帧组及每一帧效果
 * 
 * 当跟 steps() 一起使用时，@keyframes 作用于: 设置关键帧
 */

eg: 

@keyframes mymove {

  0%   {top: 0px;}
  25%  {top: 200px;}
  50%  {top: 100px;}
  75%  {top: 200px;}
  100% {top: 0px;}
}

@keyframes mymove {

  from   {top: 0px;}
  25%  {top: 200px;}
  50%  {top: 100px;}
  75%  {top: 200px;}
  to {top: 0px;}
}
```

#### Steps Animation

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * steps(num, direction);
 * 
 * 
 * @num: 每两帧之间将添加多少过渡帧;
 * 
 * @direction: start |                      // 保留下一帧状态，直到这段动画时间结束
 * 
 *              end;                        // 保留当前帧状态，直到这段动画结束
 */

steps(1, end);   == step-end

steps(1, start); == step-start


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
