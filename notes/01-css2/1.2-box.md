#### inline

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * display: inline;
 * 
 * 
 * 无法设置 width 和 height，盒子大小随着内容变化
 * 
 * 依然可以设置 margin, padding, border 但不占据垂直方向的空间，仅扩展背景
 * 
 * margin, padding, border 仅横向占用空间有效
 */

              margin                              margin                              margin               
+ ------------------------------- + + ------------------------------- + + ------------------------------- +
|             border              | |             border              | |             border              |
|   + ----------------------- +   | |   + ----------------------- +   | |   + ----------------------- +   |
|   |         padding         |   | |   |         padding         |   | |   |         padding         |   |

|   |   | \\\ content \\\ |   |   | |   |   | \\\ content \\\ |   |   | |   |   | \\\ content \\\ |   |   |
|   |   + --------------- +   |   | |   |   + --------------- +   |   | |   |   + --------------- +   |   |
```
